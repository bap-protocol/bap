# @bap-protocol/spec

JSON Schemas and TypeScript types for the Browser Agent Protocol.

This package is the canonical source of truth for the protocol data shapes. Every other package — `@bap-protocol/core`, `@bap-protocol/compliance`, and third-party implementations — imports types and validators from here.

## Contents

- `src/` — TypeScript type definitions (one file per primitive)
- `schemas/` — JSON Schema (draft 2020-12) files, published alongside the built package

## Status

Pre-alpha. APIs change daily.
